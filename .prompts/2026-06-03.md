- After the number of cycles has finished, issue a "CALL 0" OR "RST 0" instruction. This allows to synchronize the emulator with the end of the program.
- When you have modified the demos and tests with this check, you can run the cap32 emulator with the -a option and run the program, then do an event WAITFORBREAK then an EXIT event (check the documentation and skill for the correct syntax).
